Fully asynchronous Retrieval Augmented Generation pipeline that acts as an intelligent QA system. Using asyncio, aiofiles, and httpx, it concurrently loads data while validating the structure via Pydantic. Instead of simple keyword matching, it generates text embeddings via API, uses a semaphore to safely control concurrent traffic, and applies NumPy to find the best match using cosine similarity. The pipeline then feeds this retrieved context to a large language model, which synthesizes a precise and natural response for the user.
